# Youssofal Mtplx

## How do I install these formulae?

`brew install youssofal/mtplx/<formula>`

Or `brew tap youssofal/mtplx` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "youssofal/mtplx"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
